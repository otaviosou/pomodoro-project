<h1>POMODORO-PROJECT</h1>

<h2>Sobre o projeto</h2>

<p>Pomodoro é uma técnica de estudos focada em repetições de tempo. Nela são atribuídos o tempo de 25 minutos para cada sessão de estudos e 5 minutos para um descanso. Com o pomodoro podemos ganhar uma maior produtividade porque é usado o foco total nas sessões de estudos e é dada uma distração a cada vez que conseguimos finalizar uma sessão.</p>

<img src="https://github.com/otaviosou/pomodoro-project/blob/master/screen.png?raw=true">


<h2>Por dentro do projeto</h2>

<p>O Pomodoro-project é um protótipo de uma SPA - Single Page Application - utilizando javascript puro (Vanilla). A modularização usada é do sistema de módulos do ES6, já a ferramenta de criação de build foi o Webpack.</p>


<h2>Rodando o Projeto</h2>

<p>Primeiramente é necessário ter o NodeJs instalado na máquina, se não o tiver é só clicar <a href="https://nodejs.org/en/download/">aqui</a>.
  Depois de instalado entre na pasta do projeto e use o comando <strong>npm install</strong> para baixar todas as dependências necessárias. Após isso, efetue o comando <strong>npm start</strong> para rodar o projeto.
</p>
